# Smart Charging Algorithms for Electric Vehicles

## Overview
This project implements smart charging algorithms for electric vehicles using MATLAB App Designer.
The goal is to analyze different charging scenarios and improve charging efficiency under varying
vehicle and station conditions.

## Project Scope
- Scenario-based smart charging logic
- Evaluation of charging behavior based on system parameters
- Visualization of results through the application interface

## How to Run
- Open the `.mlapp` file in MATLAB (App Designer)
- Click **Run** to start the application

## Tools & Technologies
- MATLAB
- MATLAB App Designer
